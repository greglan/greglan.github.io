## Introduction
* FPGA: Field Programmable Gate Array
* VHDL: Very High Speed Description Language


## Components
* DSP cores
* Slices, CLBs, IOBs
* CLB: Configurable Logic Blocks. Implement the logic
* LUT: Look Up Table.
* FF: Flip-Flop. Can also be used as latches
* DCM blocks: Digital Clock Management
